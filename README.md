**Sales Analysis Dashboard – Power BI**

**Project Overview**

This project presents an interactive Restaurant Sales Analysis Dashboard built using Microsoft Power BI to analyze restaurant performance, sales trends, and customer behavior.

The dataset was first cleaned and prepared using Microsoft Excel, and then analyzed and visualized using Power BI to generate meaningful business insights.

The dashboard helps understand revenue trends, popular menu items, and overall restaurant performance.

**Project Objectives**

Analyze restaurant sales performance

Identify top-selling menu items

Track sales trends over time

Understand category-wise revenue contribution

Support data-driven decisions for restaurant management

**Project Tools**

**1. Microsoft Excel**

Data cleaning

Data preparation

**2️. Microsoft Power BI**

Data modeling

DAX measures

Data visualization

Interactive dashboard creation

**Dashboard Features**

Total Sales and Revenue KPIs

Category-wise sales analysis

Top-selling menu items

Sales trend over time
